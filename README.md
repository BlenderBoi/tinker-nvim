Keymap Remapping:

1. Indent Shift will Stays in Visual Mode

   - Visual: Shift-< | Shift->

2. Window Navigation

   - Normal: Ctrl-H/J/K/L

3. Window Resize

   - Normal: Ctrl-ArrowKeys

4. Buffer Navigation

   - Normal: Shift-H/L
